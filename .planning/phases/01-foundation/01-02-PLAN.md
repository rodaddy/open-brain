  4. If match found, set `(req as any).auth = matchedAuthInfo`, call `next()`
  5. If no match, return 401 JSON `{ error: "Invalid token" }`
  6. If no Authorization header or wrong scheme, return 401 JSON `{ error: "Missing Bearer token" }`

- Import `logger` from `./logger.ts`
- Define `EMBEDDING_TIMEOUT_MS = 5000`
- Export `generateEmbedding(text: string, litellmUrl?: string): Promise<number[] | null>`:
  1. Use `litellmUrl` parameter (defaults to `process.env.LITELLM_URL`) -- this enables testing without env vars
  2. Create `AbortController` with 5s timeout via `setTimeout`
  3. POST to `${litellmUrl}/embeddings` with JSON body: `{ model: "embeddings", input: text, dimensions: 768 }`

- Export `contentHash(text: string): string`:
  1. Normalize: `text.trim().replace(/\s+/g, " ")`
  2. Return `createHash("sha256").update(normalized).digest("hex")`
